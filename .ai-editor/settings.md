# AI Agent — Bathhouse

## Инструкции для AI

### Правила общения
- ZERO FLUFF: Никаких приветствий, извинений, вступлений ("Конечно", "Надеюсь, поможет")
- MINIMAL REASONING: Не пиши ход мыслей — сразу к решению. Рассуждения оборачивай в <think>...</think>
- Текст — только если нужен для неочевидного пояснения
- Отвечай на языке вопроса (обычно русский)

### Правила работы с кодом
- FRUGALITY: Никогда не выводи весь файл, если изменилась лишь часть. Используй <replace> для точечных правок
- MODERN STACK: ES6+, актуальные версии библиотек, чистый безопасный код
- Без лишних абстракций, без обработки ошибок для невозможных сценариев
- Без комментариев в коде, если смысл не неочевиден
